# Backend

Aquí vivirá el backend de Magnus (ASP.NET Core Web API), que reemplazará al
mock server de json-server cuando esté listo.

- API REST bajo el prefijo `/api/v1`.
- Envelope de éxito: `{ "data": ..., "meta": { ... } }`.
- Autenticación con JWT Bearer.
